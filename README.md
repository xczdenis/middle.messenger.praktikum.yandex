## Установка и запуск
Сервис запускается на 3000 порту.

1. Клонируйте репозиторий: `git clone`
2. Установите зависимости: `npm install`
3. Сборка: `npm run build`
4. Запуск dev сервера: `npm run serve`
5. Запуск prod сервера: `npm run start`
